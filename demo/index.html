Truy vấn phân tán

Sản phẩm còn hàng tại kho miền Bắc. Cục bộ hóa hoàn toàn tại S1, chi phí truyền thông = 0.

-- Thực thi tại Site 1 (cục bộ hoàn toàn)
SELECT SP.TenSP, TK.SoLuongTon
FROM SP_PUBLIC SP
JOIN TONKHO_MB TK ON SP.MaSP = TK.MaSP
JOIN KHO_MB K ON TK.MaKho = K.MaKho;

Tổng tồn kho SP001 trên toàn hệ thống. Tổng hợp cục bộ tại mỗi trạm, truyền 3 giá trị số.

-- Tổng hợp cục bộ: mỗi trạm gửi 1 giá trị
SELECT SUM(TonCucBo) AS TongTonKho
FROM (
    SELECT SUM(SoLuongTon) AS TonCucBo FROM TONKHO_MB WHERE MaSP = 'SP001'
    UNION ALL
    SELECT SUM(SoLuongTon) FROM TONKHO_MT WHERE MaSP = 'SP001'
    UNION ALL
    SELECT SUM(SoLuongTon) FROM TONKHO_MN WHERE MaSP = 'SP001'
) AS KetQua;

Đơn hàng miền Nam xuất từ kho miền Bắc (chéo vùng). Phép nối nửa: chỉ truyền tập khóa MaKho từ S1.

-- Semijoin: chỉ truyền π(MaKho) từ S1 → S3
SELECT DISTINCT D.MaDH
FROM DONHANG_MN D
JOIN CTDH_MN C ON D.MaDH = C.MaDH
WHERE C.MaKhoXuat IN (
    SELECT MaKho FROM KHO_MB
);

Doanh thu theo kho trong tháng hiện tại. Tổng hợp cục bộ 2 giai đoạn: GROUP BY tại mỗi trạm, gom cuối tại điều phối.

-- Gom nhóm cục bộ tại 3 trạm → tổng hợp cuối
SELECT MaKhoXuat, SUM(DoanhThuCucBo) AS DoanhThu
FROM (
    SELECT C.MaKhoXuat, SUM(C.SoLuong * C.DonGia) AS DoanhThuCucBo
    FROM CTDH_MB C JOIN DONHANG_MB D ON C.MaDH = D.MaDH
    WHERE MONTH(D.NgayLap) = MONTH(GETDATE())
    GROUP BY C.MaKhoXuat
    UNION ALL  -- ... S2 + S3 tương tự
) AS TongHop
GROUP BY MaKhoXuat;

Sản phẩm bán chạy nhất toàn hệ thống. Nhân bản SP_PUBLIC + tổng hợp cục bộ.

-- Tổng hợp MaSP cục bộ → nối SP_PUBLIC cuối
SELECT SP.TenSP, SUM(BanCucBo) AS TongBan
FROM (
    SELECT MaSP, SUM(SoLuong) AS BanCucBo FROM CTDH_MB GROUP BY MaSP
    UNION ALL
    SELECT MaSP, SUM(SoLuong) FROM CTDH_MT GROUP BY MaSP
    UNION ALL
    SELECT MaSP, SUM(SoLuong) FROM CTDH_MN GROUP BY MaSP
) AS T JOIN SP_PUBLIC SP ON T.MaSP = SP.MaSP
GROUP BY SP.TenSP ORDER BY TongBan DESC;
